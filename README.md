# 🔥 FIT27 — Telegram Bot

Бот для трекинга похудения до 27 мая.

## Функции
- ✅ Ежедневный чек-ин (вес, шаги, тренировка, оценка)
- 🌙 Трекер сна (время засыпания/пробуждения, оценка качества)
- 📅 Визуальный календарь
- 📈 Аналитика и графики
- 🎮 Геймификация (XP, уровни, достижения, челленджи)
- 💾 Данные хранятся на сервере — не пропадают

## Деплой на Railway (бесплатно)

### Шаг 1 — GitHub
1. Создай аккаунт на github.com
2. Создай новый репозиторий (New repository → назови fit27bot → Create)
3. Загрузи все файлы: bot.py, requirements.txt, Procfile

### Шаг 2 — Railway
1. Зайди на railway.app
2. Нажми "Start a New Project"
3. Выбери "Deploy from GitHub repo"
4. Выбери свой репозиторий fit27bot

### Шаг 3 — Переменные окружения
1. В Railway открой свой проект
2. Зайди в Variables
3. Добавь переменную:
   - Name: BOT_TOKEN
   - Value: твой токен от BotFather

### Шаг 4 — Запуск
Railway сам запустит бота. Через 1-2 минуты он будет работать!

## Команды бота
- /start — настройка целей
- /checkin — чек-ин на сегодня
- /sleep — записать сон
- /home — главный экран
- /calendar — календарь
- /analytics — аналитика
- /game — уровни и достижения
- /settings — настройки и экспорт данных

## Данные
Данные хранятся в файле data.json на сервере Railway.
Используй /settings → "Экспорт данных" для резервной копии.
